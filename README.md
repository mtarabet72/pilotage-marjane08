# Pilotage Hypermarché — Direction

PWA de pilotage interne Marjane (Tanger), déployée via GitHub Pages.

## Structure
```
/
├── index.html
├── manifest.json
├── service-worker.js
└── icons/
    ├── icon-192-marjane.png
    └── icon-512-marjane.png
```

## Déploiement GitHub Pages

1. Créer un repo GitHub (ex: `pilotage-marjane`)
2. Uploader tous ces fichiers à la racine
3. Aller dans **Settings → Pages → Source → main / root**
4. L'app sera disponible sur : `https://<username>.github.io/pilotage-marjane/`
